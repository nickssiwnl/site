resumo de HTML

Resumo sobre a linguagem HTML História do HTML: HTML, ou HyperText Markup Language, é a linguagem padrão utilizada para criar e estruturar páginas web.
Foi criado por Tim Berners-Lee, um cientista britânico, em 1991.
Tim Berners-Lee trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear) e desenvolveu HTML como parte de um projeto para facilitar o compartilhamento de documentos científicos.
A primeira versão do HTML, conhecida como HTML 1.0, foi lançada em 1993.
Desde então, a linguagem evoluiu significativamente com várias atualizações e melhorias.
As versões mais notáveis incluem HTML 4.01, lançada em 1999, e HTML5, que foi padronizada em 2014.
HTML utiliza uma série de "tags" para definir diferentes elementos em uma página web.

Aqui está um exemplo básico de um documento HTML:


html

< !DOCTYPE html>
< html lang="pt-BR">
< head>
< meta charset="UTF-8">
< meta name="viewport" content="width=device-width, initial-scale=1.0">
< title>Título da Página< /title>
< /head>
< body>
< h1>Olá, Mundo!< /h1>
< p>Este é um parágrafo em HTML.< /p>
< /body>
< html>

Principais Tags do HTML:


< html>: Define o documento como um documento HTML.
< head>: Contém metadados sobre o documento, como título e configuração de caracteres.
< title>: Define o título da página, exibido na aba do navegador.
< body>: Contém todo o conteúdo visível da página, como textos, imagens e links.
< h1> a < h6>: Definem cabeçalhos, onde < h1> é o mais importante e < h6> o menos importante.
< p>: Define um parágrafo.
< a>: Cria um link.
< img>: Insere uma imagem.

Importância do HTML: HTML é fundamental para a construção de sites e aplicações web. Ele permite que desenvolvedores criem estruturas e conteúdos de forma organizada e acessível.


Junto com CSS (Cascading Style Sheets) para estilos visuais e JavaScript para funcionalidades dinâmicas, HTML compõe o núcleo do desenvolvimento web.


Evolução do HTML5: HTML5 trouxe várias melhorias e novas funcionalidades, incluindo:


Suporte nativo para áudio e vídeo.


Novos elementos semânticos como < article>, < section>, < header>, e < footer>.


Melhor suporte para gráficos e animações com < canvas> e SVG.


APIs para interação avançada com o usuário e armazenamento local.


HTML continua a evoluir, e a comunidade web trabalha constantemente para aprimorar a linguagem e adaptá-la às necessidades modernas da web.
